## 📋 Resumen

Este repositorio contiene la implementación numérica completa y los resultados del **Marco de Campos Trinitarios**, una teoría unificadora que resuelve **17 paradojas fundamentales** de la física contemporánea mediante **solo 4 postulados y 2 parámetros**.

### La idea central

Toda entidad física es un **Campo Trinitario**:

**Ψ = Pf ⊗ Pi ⊗ Pd**

donde:
- 🔷 **Pf (Forma)**: Información geométrica, núcleo estable
- 💚 **Pi (Líquido/Intención)**: Energía-información contenida
- 🔴 **Pd (Dinámico)**: Onda, oscilación, proyección radiante

La **Gradiente de Acoplamiento Γ** mide la cohesión entre estos componentes:

- **Γ ≪ 1** → Régimen cuántico (onda, fotón)
- **Γ = 1** → Frontera micro-macro
- **Γ ≫ 1** → Régimen clásico (partícula, protón)
